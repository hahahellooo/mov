# mov

### install
```bash
# main
$ pip install git+https://github.com/hahahellooo/mov.git

# branch
$ pip install pip install git+https://github.com/hahahellooo/mov.git@
```

### start dev
```bash
$ git clone <URL>
$ cd <DIR>
$ source .venv/bin/activate
$ pytest

$ # option
$ pdm venv create
```

### setting env
```bash
$ cat ~/.zshrc | tail -n 3
```
